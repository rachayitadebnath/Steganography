Zero-Steganography

A secure image hiding system that transfers secret images without modifying the cover image. The system generates a stego-key instead of embedding data, making it undetectable by steganalysis.

Sender Mode

Receiver Mode

Upload the cover image and stego-key to recover the original secret image.

How It Works

Unlike traditional steganography that embeds data into images, zero-steganography generates a mathematical key from the difference between cover and secret images. This key, combined with the original cover image, can perfectly reconstruct the secret image.
